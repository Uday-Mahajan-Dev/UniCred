# 🚀 UniCred  
### Universal Credential Intelligence Platform

![Flutter](https://img.shields.io/badge/Flutter-App-blue?logo=flutter)
![Firebase](https://img.shields.io/badge/Firebase-Backend-orange?logo=firebase)
![AI Powered](https://img.shields.io/badge/AI-Powered-purple)

![License](https://img.shields.io/badge/License-MIT-brightgreen)

**UniCred** is a next-generation **digital credential ecosystem** that enables secure issuance, verification, and intelligent analysis of certificates across industries.

It connects **Citizens, Issuers, Employers, and Administrators** into a unified platform where credentials become **verifiable, intelligent, and career-driven assets**.

# 🧠 AI Powered Features

UniCred integrates AI to transform static certificates into **intelligent career assets**.

### ✨ AI Capabilities

- Credential analysis  
- Skill extraction from certificates  
- Career path recommendations  
- Certification suggestions  
- Skill gap detection  

---

# 📄 AI Resume Builder (ATS Friendly)

Citizens can generate **AI-powered resumes** using:

- Stored certificates  
- Skills extracted from credentials  
- Additional details entered by the user  

The system automatically generates:

- ATS-friendly resume formats  
- Structured professional profiles  
- Skill-based resumes optimized for hiring platforms  

---

# 👥 Platform Roles

## 👤 Citizen

The Citizen dashboard acts as a **digital credential wallet**.

### Features

- Store and manage credentials  
- AI analysis of certificates  
- Career recommendations  
- Skill gap identification  
- Generate **ATS-friendly AI resumes**  
- Share credentials with employers instantly  

---

## 🏢 Issuer (Organizations / Institutions)

Issuers can create and manage **digital certificates**.

### Features

- Issue digital credentials  
- Manage certificate records  
- Track issued certificates  
- Monitor credential verification activity  
- **Analytics dashboard for issuance insights**

---

## 💼 Employer

Employers can verify and evaluate credentials before hiring.

### Features

- Search candidates by skills  
- Verify certificates instantly  
- View credential authenticity  
- Access verified candidate profiles  

---

## 🛠 Admin

The Admin role ensures ecosystem governance.

### Features

- Platform monitoring  
- Issuer management  
- Credential ecosystem control  
- **Analytics dashboard for system activity**

---

# 📊 Analytics Dashboard

Both **Issuers and Admins** have access to analytics to monitor activity.

### Metrics Include

- Total certificates issued  
- Verification activity  
- Credential distribution  
- User engagement statistics  
- Issuer performance insights
